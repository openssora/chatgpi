ChatGPT 服务降级检测扩展程序

由于 ChatGPT 会对某些 ip 进行无提示的服务降级，此扩展程序用于检测你的 ip 是否被 ChatGPT 判定为高风险。在一定程度上可以用于辅助判断你的 ip 是否遭到服务降级。

（1）点击添加至Chrome就安装成功了

（2）然后打开chatgpt就直接可以看到页面上多了一个小绿圈, 鼠标移上去就可以看到当前的PoW信息。

如果 PoW 难度的值很低，代表你的 ip 可能被判断为了高风险。 作为参考，这个值在超过 5 位时，一般代表你的ip较为优质，可以正常使用所有服务，如果小于等于 000032，说明你的 ip 被认为有很高的风险。

你每次登陆或者刷新ChatGPT时，小绿圈都会重新监测当前的pow。每次发现GPT回复消息特别快时，都可以看一下，非常方便的辅助你判断当前的IP环境。

  • 当IP质量高风险时，小绿圈就被成小红圈。
  • 当IP质量中风险时，小绿圈就变成小黄圈。
